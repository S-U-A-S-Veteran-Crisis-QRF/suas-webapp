# Definition of done — SUAS grant-prospect report

The run is satisfied only if ALL of the following are true:

1. `grant-report.md` exists and lists **at least 8** currently-open funding opportunities relevant to veteran crisis, mental-health, suicide-prevention, or emergency/quick-reaction work.
2. Each opportunity includes: **funder name**, **award amount or range**, **deadline** (or "rolling"), an **eligibility summary**, a one-line **"why it fits SUAS"**, and a **working application/source URL the agent actually retrieved**.
3. The **top 3** best-fit opportunities each include a tailored **letter-of-inquiry / outreach email draft of 250 words or fewer**, addressed to that funder.
4. **Nothing is fabricated** — no funder, link, deadline, or amount that wasn't confirmed from a fetched source. Anything unverifiable is omitted or explicitly marked "unverified".
5. Opportunities are **ranked** by fit + award size + deadline proximity, with a one-line ranking rationale.
